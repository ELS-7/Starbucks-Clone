과목추가


1. 학생추가 : 이름, 열 개수만큼 성적 입력하도록 표기.
    어떤 과목 점수를 입력해야하는지 표기됨.
    양식(입력 과목 개수)가 잘못됐다면 오류 출력.
    이미 있는 이름이라면- 해당 번호와 동명이인에 주의하라는 문구 출력.

2. 학생목록
    양식에 맞춰 출력됨

3. 성적변경 : 사람번호와 과목종류, 새 점수 입력시 수정됨
    없는 번호(사람번호 or 과목번호) 입력시 오류표기

4.학생삭제
- 없는 번호 삭제 시도시 없음 표기

5. 학생검색 : 이름 - 키워드(일부 가능)로 검색시 목록 표기
